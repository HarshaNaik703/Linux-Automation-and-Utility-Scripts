# Git-Related Commands

This folder contains the files which helps to automate git scripts

1. git-init -> This helps to initiate the git in your folder. execute this command, give your file name, add your remote repo link and you are done!!!
2. git-automation -> This file helps you to add the files you want to add and choose the branch , and commit message and you're done again!!!
3. git-track-deleted-files -> This file will help you get rid of files which are pushed in remote repo but doesn't exit in local repo
